# Effect Studies
This repository contains data analysis reports and code for various effect studies conducted in R languange with Quarto framework, focusing on the financial markets. The studies delve into anomalies, patterns and inefficiencies, employing data analysis techniques. Each report offers:
    - Clear explanations: concise breakdowns of research methods, findings and conclusions
    - Reproducible codeL corresponding R code for each study, ensuring transparancy and facilitating understanding.
